# Security Policy

Report vulnerabilities privately via
[GitHub Security Advisories](https://github.com/Viprasol-Tech/react-resizable-panels/security/advisories/new)
or [support@viprasol.com](mailto:support@viprasol.com). We aim to respond within
72 hours. Never commit secrets; rotate anything exposed.
