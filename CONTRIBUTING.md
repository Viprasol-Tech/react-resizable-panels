# Contributing to react-resizable-panels

Thanks for your interest! This project is maintained by
[Viprasol Tech](https://viprasol.com).

## Setup

```bash
git clone https://github.com/Viprasol-Tech/react-resizable-panels.git
cd react-resizable-panels
npm install
```

## Before opening a PR

```bash
npm run typecheck   # tsc --noEmit
npm test            # vitest run
```

- Keep everything strictly typed (TypeScript `strict` is on).
- Add tests for new behaviour.
- Never commit secrets or `node_modules`.
